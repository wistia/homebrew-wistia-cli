# Wistia Wistia-cli

## How do I install these formulae?

`brew install wistia/wistia-cli/<formula>`

Or `brew tap wistia/wistia-cli` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "wistia/wistia-cli"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
